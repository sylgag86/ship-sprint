# Chapter 9 — Operating Rules (Never Break)

1. One offer per sprint.
2. No rebuilding mid-sprint.
3. No design rabbit holes before checkout works.
4. Distribution volume beats overthinking.
5. First sale > first logo polish.

## Failure Modes to Avoid

- Endless idea switching
- Waiting for perfect tools
- Posting once and calling it a launch
- Not following up warm leads

## Success Rule

You are judged by shipped offers and close attempts, not by preparation quality.

## Final Checklist

- [ ] Offer shipped
- [ ] Checkout tested
- [ ] Assets published
- [ ] Outbound sent
- [ ] Follow-ups done
- [ ] Proof captured

Run it. Measure it. Improve it. Repeat.
