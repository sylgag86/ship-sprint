# Chapter 2 — Choose the Right Product in 45 Minutes

If you choose the wrong product, execution speed won’t save you. This chapter gives you a strict filter so you can decide fast and move.

## The 3-Fit Filter

Score each idea from 1–5 on:

1. **Pain Intensity** — Does this problem hurt enough to pay now?
2. **Buyer Liquidity** — Does this audience currently spend on this type of solution?
3. **Delivery Speed** — Can you deliver the first version inside 24 hours?

Any idea scoring below 10 total is cut.

## Fast Candidate Generation (10 minutes)

Generate 5 offer ideas max. Keep each idea in one line:

- Audience
- Problem
- Outcome
- Format

Example:
"Local roofers — inconsistent social posting — 30-day post kit — delivered as templates + captions"

Do not research for an hour. Generate, score, cut.

## Red Flags (Instant Disqualifiers)

Kill any idea that requires:
- multi-step onboarding before value
- custom app logic to deliver the core result
- legal/regulatory approvals before sale
- integrations that may break before launch

Sprint offers should be robust under pressure.

## Offer Sentence Test

Write one sentence for each finalist:

> We help **[audience]** get **[outcome]** in **[time]** without **[pain]**.

If the sentence sounds vague, the offer is vague.

## Price Fit Test

Run one of these fixed anchors:
- Entry digital product: $19–$49
- Template/productized bundle: $49–$99
- High-touch review/service add-on: $149–$299

If you cannot justify one anchor clearly, reframe outcome.

## 45-Minute Decision Sprint

- Minute 0–10: generate 5 ideas
- Minute 10–25: score each with 3-Fit
- Minute 25–35: write offer sentence for top 2
- Minute 35–45: pick one and lock it

No re-opening after minute 45.
