# Chapter 8 — Weekly Repeat System

The sprint becomes powerful when repeated weekly.

## Weekly Cadence

- Day 1: run sprint on current offer angle
- Day 2: review objections and conversion data
- Day 3: improve page and scripts
- Day 4: relaunch
- Day 5: follow-up and close

## What to Improve Each Cycle

1. Headline clarity
2. Offer positioning
3. Pricing stack
4. DM open rate
5. Follow-up speed

## KPI Benchmarks

- Reply rate target: 15–30%
- Checkout click-through: 5–12%
- First-week close rate: 1–5%

If below benchmarks, tighten audience and simplify promise.

## Chapter 8 Checklist

- [ ] Weekly sprint date scheduled
- [ ] Data from prior cycle logged
- [ ] One major variable changed
- [ ] Relaunch executed
